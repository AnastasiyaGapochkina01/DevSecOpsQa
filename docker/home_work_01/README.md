# Docker Learning Project

A comprehensive Docker learning project covering containerization fundamentals to multi-service applications.

## Projects

### Task 1: Single Nginx Container
- Simple web server with volume mount
- Port mapping: 8080:80
- Local HTML file mounting

### Task 2: Single Flask Container  
- Python Flask application
- Log volume persistence
- Port mapping: 5000:5000

### Task 3: Flask + Redis (Docker Compose)
- Multi-container application
- Redis for session storage
- Docker Compose orchestration
- Volume persistence

### Task 4: WordPress + MySQL (Docker Compose)
- Full CMS deployment
- MySQL database with persistent volume
- WordPress with persistent files
- Docker Compose orchestration

## Prerequisites
- Docker
- Docker Compose

## Usage
See individual task directories for specific instructions.

## Learning Outcomes
- Docker container basics
- Volume mounting for persistence
- Port mapping for network access
- Dockerfile creation
- Docker Compose for multi-container apps
- Container networking
- Resource management
